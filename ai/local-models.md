# Local Models

Links of tools to download and run local models.

## Tools

- [Can I Run AI](https://www.canirun.ai/)
  Checks your system and tells you which models you could run locally and how good.

- [Llama.cpp](https://llama-cpp.com/)
  Llama.cpp (LLaMA C++) allows you to run efficient Large Language Model
  Inference in pure C/C++. You can run any powerful artificial intelligence
  model including all LLaMa models, Falcon and RefinedWeb, Mistral models,
  Gemma from Google, Phi, Qwen, Yi, Solar 10.7B and Alpaca.

  With this, you can run a model in your CPU, no need for GPU. Although you'd
  probably not be able to run the biggest models or not run at a fast pace.

- [Ollama](https://ollama.com/)
  Ollama is a local runtime and model management tool for large language
  models: it lets you download, run, and interact with LLMs on your own machine
  through a simple CLI or API, abstracting away setup details like model
  formats, inference backends, and hardware configuration, so you can treat
  models almost like local services (pull → run → query) without manually
  wiring together tools such as llama.cpp, GPU settings, or prompt templates.

  **So, it seems with this I don't need to use Llama.cpp.**

- [LM Studio](https://lmstudio.ai/)
  Same as Ollama: it easis the management and running of local models.

## Models

- [HuggingFace](https://huggingface.co/)
  A huge site with lots of information and also the place from where one
  downloads models. However, instead of downloading via web interface, use
  `ollama` to get your models.
