# Design

- [Impeccable](https://impeccable.style/)
  Probably the only link this section will ever have, but who knows.

  Impeccable is a set of rules in an npm package to fix designs produced by
  AIs. It seems, I haven't tried yet, that AIs produce very repeated patterns
  when you use them to design web sites. This package helps audit and correct
  such designs.
