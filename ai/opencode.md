# OpenCode

## Plugins

- [oh-my-opencode](https://github.com/code-yeongyu/oh-my-openagent)
- [omo site](https://ohmyopenagent.com/)
  A set of agents with different pre-configured models to use the propel model
  to each task and let the AI develop code alone.

  - 2026-04-05 just installed it today. Haven't done anything with it yet.

- [OpenCode Magic Context](https://github.com/cortexkit/opencode-magic-context)
  An auto-compacter that let's the AI continue working as it manages the
  compacting in the backgroun and, I think, preemptively. Haven't tested it yet,
  just installed omo and I wanna make a test run of it without any other extras
  before.

- [AFT](https://github.com/cortexkit/aft)
  Let AIs use AST instead of reading code files as text. Duh!
